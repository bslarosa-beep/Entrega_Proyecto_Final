# Pre-Entrega proyecto - SauceDemo

Este repositorio contiene un conjunto de pruebas de automatización (E2E) para el sitio web [SauceDemo](https://www.saucedemo.com/). El proyecto está construido con Python, Selenium y Pytest, y se enfoca en validar los flujos de inicio de sesión y la funcionalidad básica del inventario.

---

## Tecnologías Utilizadas

* **Python 3.14.1
* **Selenium:** Para la automatización del navegador.
* **Pytest:** Como framework para la ejecución de pruebas y manejo de fixtures.
* **ChromeDriver:** Como el driver específico para Google Chrome.

---

##  Prerrequisitos

Antes de empezar, asegúrate de tener instalado:
* [Python 3](https://www.python.org/downloads/)
* [Google Chrome](https://www.google.com/chrome/)
* Tener **ChromeDriver** en el PATH del sistema o en la carpeta del proyecto.
---

### 1. Ejecutar todas las pruebas

Para ejecutar el conjunto completo de pruebas, simplemente corre:
pytest -v
